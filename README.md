# 🛍 OGR Store — Telegram Mini App

Премиальная витрина магазина техники, открывается прямо внутри Telegram.
Тёмная тема **black + gold**, каталог сеткой, фильтры, карточка товара,
корзина, избранное, поиск, оформление заказа и блок доверия.

**Стек:** чистый HTML/CSS/JS (один файл `index.html`, без сборки).
**Чекаут:** визуальный (показывает подтверждение в приложении, без отправки).

---

## 👀 Быстрый просмотр

Просто откройте `index.html` в браузере двойным кликом — всё работает
(вне Telegram тактильный отклик и кнопка «назад» Telegram недоступны, остальное — да).

## 🚀 Запуск как Mini App (3 шага)

### Шаг 1 — Захостить `index.html` (нужен HTTPS)
Любой бесплатный статический хостинг. Самое простое:

**GitHub Pages:**
1. Создайте репозиторий, залейте файлы (`index.html` + папку `assets/`)
2. Settings → Pages → Branch: `main` → Save
3. Получите адрес вида `https://username.github.io/ogrstore/`

**Или Netlify / Vercel:** перетащите папку — получите HTTPS-ссылку за минуту.

> ⚠️ Telegram открывает Mini App только по **HTTPS**. Локальный `file://` не подойдёт для запуска внутри Telegram.

### Шаг 2 — Привязать к боту через @BotFather
1. `/newbot` → получите токен (если бота ещё нет)
2. `/newapp` → выберите бота → задайте название, описание, иконку
3. Вставьте **URL из шага 1** как Web App URL
4. (Альтернатива) `/setmenubutton` → выберите бота → вставьте URL → задайте текст кнопки «Магазин»

Готово — синяя кнопка-меню слева от поля ввода открывает витрину.

### Шаг 3 (опционально) — Бот-лаунчер с кнопкой
Если хотите, чтобы по `/start` приходило приветствие с кнопкой «Открыть магазин»:
```bash
pip install -r requirements.txt
cp .env.example .env        # впишите BOT_TOKEN и WEBAPP_URL
python launcher_bot.py
```

## ✏️ Как добавить / изменить товар

Откройте `index.html`, найдите массив `PRODUCTS` (вверху `<script>`) и допишите:
```js
{id:"ip17-256", cat:"iphone", model:"iPhone 17", variant:"256 GB",
 color:"Чёрный", price:99990, old:0, stock:true, hit:false, photo:""},
```
- `cat` — ключ из `CATEGORIES` (iphone / macbook / ipad / samsung / ps5 / watch / acc)
- `old` — старая цена для бейджа скидки (0 = нет)
- `stock` — true (в наличии) / false (под заказ)
- `hit` — true показывает золотой бейдж «Хит»
- `photo` — путь к фото или `""` (тогда — плейсхолдер `assets/products/{id}.svg`)

---

## 🖼 Фото товаров
Под каждый товар уже сгенерирован брендовый плейсхолдер в `assets/products/{id}.svg`
(тёмный фон + золотая иконка категории + подпись). Витрина выглядит цельно сразу.

Когда появятся реальные фото:
1. Положите файл в `assets/products/` (например `ip16p-256.jpg`)
2. Пропишите путь в поле `photo` в массиве `PRODUCTS`
3. Пустое `photo:""` → используется плейсхолдер

Перегенерировать плейсхолдеры: `python3 gen_placeholders.py`

## 🎨 Тема и брендинг
- **Вся тема — в `theme.css`** (CSS-переменные: фон, золото, текст, отступы, шрифты).
  Перекрасить магазин = поменять только этот файл. Задел под светлую тему уже описан внутри.
- Логотип в шапке и на главной — иконка Lucide в золотой рамке. Чтобы поставить
  ваш PNG-логотип: замените `<i data-lucide="smartphone">` в `.logo-frame` на `<img src="assets/logo.png">`.
- Иконки — [Lucide](https://lucide.dev) через CDN (атрибут `data-lucide="имя"`).
- Шрифты: Playfair Display (заголовки) + Jost (интерфейс), Google Fonts.

## 📁 Структура
```
ogrstore_miniapp/
├── index.html          ← приложение (данные + макет + логика)
├── theme.css           ← тема: цвета, шрифты, токены
├── launcher_bot.py     ← бот, открывающий витрину (опционально)
├── gen_placeholders.py ← генератор фото-плейсхолдеров
├── .env.example        ← шаблон токена и URL
├── requirements.txt
└── assets/
    └── products/       ← фото/плейсхолдеры товаров ({id}.svg)
```
> ⚠️ При хостинге заливайте `index.html` **вместе с** `theme.css` и папкой `assets/` — иначе слетит тема и пропадут картинки.
